Firebase e historico online

Arquivos principais alterados:

- index.html
- style.css
- firebase.js
- script.js
- historico.js

O projeto ja contem a configuracao Firebase informada no pedido.
Para funcionar online, publique estes arquivos em um servidor HTTPS ou no GitHub Pages e mantenha o Firestore Database ativo no projeto atlas-bobina.

Colecoes usadas no Cloud Firestore:

- usuarios
- calculos_bobina
- calculos_agropainel
- historico_bobina
- historico_agropainel
- testes_banco

A colecao historico_bobina registra:

- Data
- Hora
- Tipo da acao
- Usuario
- Produto/Bobina
- Quantidade
- Observacao

A colecao historico_agropainel registra:

- Data
- Hora
- Acao realizada
- Valores antes e depois da alteracao
- Usuario

O botao "Testar Banco de Dados" cria e le um documento na colecao testes_banco.
Se a leitura funcionar, a tela mostra a mensagem verde de sucesso.
Se houver erro de regra, conexao ou configuracao, a mensagem completa aparece em vermelho.
