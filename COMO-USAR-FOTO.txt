Para usar a imagem completa da maquina no visual animado:

1. Salve a foto na mesma pasta destes arquivos.
2. Renomeie a foto para maquina.jpg.
3. Abra o index.html no navegador.

O app usa a foto completa como fundo e anima apenas a bobina traseira/direita.
Se a bobina animada ficar um pouco fora do lugar, ajuste estes valores no style.css:

.bobinaAnimada {
    left: 70.7%;
    top: 42.7%;
    width: 20.5%;
}

.folhaSaindo {
    left: 66%;
    top: 66%;
}
